# Glozzom

A multi-page marketing-style website built with plain HTML and CSS, using Bootstrap for layout and components. 


## Tech stack

- **HTML5** — structure and semantics
- **CSS3** — custom styles in `style.css` (typography, colors, utilities)
- **[Bootstrap 5.3](https://getbootstrap.com/)** — grid, navbar, utilities (loaded from CDN)
- **[Font Awesome 6](https://fontawesome.com/)** — icons (CDN)
- **[Google Fonts](https://fonts.google.com/)** — Montserrat and Roboto

## How to run locally

Because this is static HTML, you can open `index.html` directly in a browser.
